# Cordova crypt file plugin
HTML source file is encrypted at build, and decrypted at run.  
https://www.npmjs.com/package/cordova-plugin-crypt-file

## Add Plugin
`cordova plugin add cordova-plugin-crypt-file`

## Encrypt
`cordova build [ios / android]`

## Decrypt
`cordova emulate [ios / android]`  
or  
`cordova run [ios / android]`  

## Encryption subjects.
* .html
* .htm
* .js
* .css

## Custom filepath pattern
Add array of filepath patterns that would be encrypted within **package.json** with key name `encFilePathPatterns`, otherwise will encrypt all script files.

## Supported platforms
* iOS
* Android
* CrossWalk
